## 🚀 About The Project

Welcome to my Python archive! This repository documents an intensive journey into Python programming. It houses hundreds of scripts ranging from foundational concepts to advanced algorithmic challenges. 

If you are a student, educator, or developer looking for structured programming problems to test or enhance your Python skills, you've landed in the right place!

### 🎯 Key Features
- **Structured Learning Path:** Separated into 11 weeks of progressive difficulty.
- **Diverse Problem Sets:** Includes Practice Programming Assignments (PPA), Graded Programming Assignments (GRPA), and general assessments.
- **Optimized Solutions:** Clean, well-documented, and efficient Python code.

## 💻 Getting Started

### Prerequisites
Make sure you have Python installed. You can download it from [python.org](https://www.python.org/downloads/).

### Installation & Execution

1. **Clone the repository**
   ```bash
   git clone https://github.com/santhoshkumar7507/Python-QUestion.git
   ```
2. **Navigate to the project folder**
   ```bash
   cd Python-QUestion
   ```
3. **Execute any script**
   ```bash
   python "Santhosh Kumar/week1/lect_1.2/ques1.2.1.py"
   ```
